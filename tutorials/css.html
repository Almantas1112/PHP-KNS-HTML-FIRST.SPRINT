CSS(How to link CSS file to HTML)

First of all we need to create index.html and style.css file

Then we format index.html file by pressing ! or !!!

in html file we need to find <head> tag and in that section we put this line in: <link rel="stylesheet" href="Path_To_Your_CSS_File/style.css">

if we have index.html and style.css files in the same directory we can adjust code and put code like this: <link rel="stylesheet" href="style.css"> Lets have a look in code:

The <link> tag defines the relationship between the current document and an external resource rel="stylesheet" this required and it specifies the relationship between the current document and the linked document href="style.css" this basically means URL, specifies how the element handles cross-origin requests. And there you have it! Now you can play with divs and have custom designed webs!
