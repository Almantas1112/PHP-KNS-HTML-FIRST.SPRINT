PHP(tags) lesson

Let's talk about PHP tags:

in HTML we have <head> and <body> tags in php we have <?php?> tag and for example it looks like this

<?php php code php code ?>

Let's have a look in real example:

<?php echo 'PHP tag'; ?> <?php is open tag for PHP and ?> is for closing tag echo means that we want to write something, for example we use <p>Something</p> in HTML and for PHP we use echo 'PHP tag' what we want to write in this message and command should always end with ;
