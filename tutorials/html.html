HTML(tags) lesson

First of all let's talk about formating new HTML file: If you using Visual Studio code (don't know if any editors do the same thing) create new file and call it index.html, then in first line press ! or !!! If you pressed ! symbol you will get code like this:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta http-equiv="X-UA-Compatible" content="IE=edge"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Document</title> </head> <body>

</body> </html>

First line shows document type Second line shows html code language third line gives <head> tag which contains information like: meta charset UTF-8 (means that it is newest charset for programming in HTML, finds all languages symbols like Ą,Č and so on) meta name viewport contains responsivity it means html site will be compactible in mobile phones, tablets and computers <title>Document</title> contains your tab name Then we have <body> tag. In this we can write all data like javascript, divs and so on.
